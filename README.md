## 🚀 Overview

**SKS Lending** is a high-performance, professional-grade management platform designed for money lenders to track loans, repayments, and borrower growth. Built with a sleek **Charcoal & Emerald** aesthetic, it provides a seamless user experience across both desktop and mobile devices.

The platform streamlines the lending process by centralizing borrower data, automating financial calculations, and providing transparent statements through unique secure links.

## ✨ Key Features

### 📊 Intelligence Dashboard
- **Real-time Analytics**: Track Total Lent, Total Repaid, and Outstanding Balance at a glance.
- **Growth Visualization**: Interactive charts showing financial trends and monthly performance.
- **Recent Activity**: A live feed of the latest transactions across the system.

### 👤 Borrower Management
- **Full lifecycle tracking**: Easily add new borrowers, update profiles, and manage active vs. completed loans.
- **Dynamic Top-Ups**: Add increments to existing loans without creating separate accounts.
- **Complete Transaction History**: A clean, card-based ledger for every payment and top-up, with easy deletion for accidental entries.

### 📱 Mobile-First Experience
- **Zero-Scroll Forms**: Intelligent, multi-column mobile layouts ensure you can complete actions without excessive scrolling.
- **Bottom-Sheet Navigation**: Native-feeling modal sheets for a thumb-friendly mobile experience.
- **Branded Splash Loader**: A custom "Hiking Arrow" transition symbolizing financial growth on every startup.

### 🔗 Public Statement Sharing
- **Secure Links**: Generate unique URLs for each borrower to view their own balance and payment history without requiring an account.
- **Transparency**: High-contrast, easy-to-read statement views for borrowers.

## 🎨 Design Philosophy

The system uses a **Premium Glassmorphism** core with a specialized **"Zinc & Emerald"** color palette:
- **Zinc Base**: Provides a sophisticated, eye-friendly dark mode.
- **Emerald Accents**: Symbolizes wealth, growth, and security.
- **Interactive Micro-animations**: Subtle hover effects and surging transitions that make the app feel alive and responsive.

## 🛠️ Technology Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Database & Auth**: Firebase (Firestore & Authentication)
- **Charts**: Recharts
- **Icons**: Lucide React

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- Firebase Account

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Empire-SK/sks-dm-money-lending.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root and add your Firebase credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   ...
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
